# Desigualdade Salarial na AdventureWorks

Projeto de análise sociológica da estrutura salarial interna de uma empresa fictícia, utilizando SQL Server e a base AdventureWorks.

## Objetivo

Investigar padrões de desigualdade, concentração de renda e hierarquização organizacional por meio da análise dos salários.

## Abordagem Teórica

- Sociologia do Trabalho
- Estratificação Social
- Marx, Weber e Bourdieu

## Base de Dados

- Microsoft AdventureWorks

## Tecnologias

- SQL Server
- Git/GitHub

## Estrutura

- `/sql` → Scripts SQL
- `/resultados` → Resultados e interpretações

## Principais Resultados

- 1,6% dos funcionários concentram ~7,7% da renda
- 20% mais ricos concentram ~39% da massa salarial
- Forte desigualdade no setor de Produção
